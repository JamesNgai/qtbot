## Qtbot

A Discord bot with some neat functionality.

## Commands

Qtbot has a lot of commands which you can find by calling `qt.help`.

Noteworthy categories are: 
- Google
- Youtube
- Weather
- Tagging system
- League of Legends
- OSRS GE Checker
- Moderator commands
- [Many more](http://definitelynaught.me)
